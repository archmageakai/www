Gacha Game

The gacha game is a game played on akai.py bot.
game commands: .gacha | .gacha_rate | .guarantee | .bag

Gachapon

Type .gacha to pull from the gachapon!
You will need to have akaiyen to do a pull. See below for more info.

Gacha Rates

Type .gacha_rate to see current rates!
Cost: 1.99 akaiyen.
Max. 5 pulls per day
(as of 3/23/25)

Guarantees

You can check how many pulls until a guarantee, using .guarantee command

It takes 25 pulls without a 5-star item/blade pulled, to get a guarantee.
(as of 3/23/25)

Check your inventory

Type .bag to check your inventory.
See player index to view inventories as well.
Refer to the index for all other data on items, blades, player inventory, etc.
